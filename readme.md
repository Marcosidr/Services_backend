# Back-end (Node.js + Express + Sequelize)

## Dependencias em uso (`package.json`)

### Producao (`dependencies`)

- `cors`: `^2.8.6`
- `dotenv`: `^17.3.1`
- `express`: `^5.2.1`
- `jsonwebtoken`: `^9.0.3`
- `pg`: `^8.20.0`
- `pg-hstore`: `^2.3.4`
- `sequelize`: `^6.37.8`

### Desenvolvimento (`devDependencies`)

- `@types/express`: `^5.0.6`
- `@types/jsonwebtoken`: `^9.0.10`
- `@types/node`: `^25.4.0`
- `nodemon`: `^3.1.14`
- `sequelize-cli`: `^6.6.5`
- `ts-node-dev`: `^2.0.0`
- `typescript`: `^5.9.3`

## Comandos para instalar as dependencias

```bash
npm install cors@^2.8.6 dotenv@^17.3.1 express@^5.2.1 jsonwebtoken@^9.0.3 pg@^8.20.0 pg-hstore@^2.3.4 sequelize@^6.37.8
npm install -D @types/express@^5.0.6 @types/jsonwebtoken@^9.0.10 @types/node@^25.4.0 nodemon@^3.1.14 sequelize-cli@^6.6.5 ts-node-dev@^2.0.0 typescript@^5.9.3
```

## Scripts uteis

```bash
npm run dev
npm run build
npm run start
```
